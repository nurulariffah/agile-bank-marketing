# Agile Bank Marketing

## Project Objective
This project develops an Agile Data Science solution to predict whether a banking customer is likely to subscribe to a term deposit.

## Dataset
The dataset used is the Bank Marketing Dataset from Kaggle. It contains customer demographic, financial, and direct marketing campaign information.

## Agile Data Science Workflow
The project follows an iterative Agile Data Science workflow:

1. Sprint 1: Dataset understanding, exploratory data analysis, and data quality assessment
2. Sprint 2: Feature engineering and baseline model development
3. Sprint 3: Improved model development using Random Forest and hyperparameter tuning
4. Sprint 4: Dashboard development, monitoring, and iterative improvement

## Models Developed
- Logistic Regression as the baseline model
- Random Forest Classifier as the improved model
